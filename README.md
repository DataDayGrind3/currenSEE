# currenSEE


## How to use this app? 
1. On the landing page, there are two buttons. 
    - Left: "Start" button
    - Right: "Help" button
2. Click on the "Start" button once to initialise the camera.
3. Click on the "Help" button to get instructions on how to use this site!
